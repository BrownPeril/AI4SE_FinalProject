关键设计决策

  • 前端端口 5173（开发）/ 80（Docker Nginx）→ API 代理到后端 8080
  • 无用户系统 — 所有 API 无需鉴权
  • H2 文件模式 — 数据持久化到文件，容器重启不丢失
  • Vite proxy — 开发时前端请求 /api 自动转发后端，解决跨域
